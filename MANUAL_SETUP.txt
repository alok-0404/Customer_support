## Option 2: Use AWS Console

1. Go to AWS EC2 Console
2. Connect to instance via Session Manager or EC2 Instance Connect
3. Run these commands:

```bash
# Install required tools
sudo yum update -y
sudo yum install -y nginx certbot python3-certbot-nginx

# Get SSL certificate
sudo certbot --nginx -d mydiamond99clientsupport.in -d www.mydiamond99clientsupport.in

# Configure Nginx manually or use the config from nginx-ssl-config.conf
```
