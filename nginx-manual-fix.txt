EC2 PAR YE STEPS FOLLOW KARO:

1. BACKUP CURRENT CONFIG:
sudo cp /etc/nginx/sites-available/mydiamond99clientsupport.in /etc/nginx/sites-available/mydiamond99clientsupport.in.backup

2. DELETE OLD CONFIG:
sudo rm /etc/nginx/sites-available/mydiamond99clientsupport.in

3. CREATE NEW FILE:
sudo nano /etc/nginx/sites-available/mydiamond99clientsupport.in

4. PASTE THIS COMPLETE CONFIG:
server {
    listen 80;
    server_name mydiamond99clientsupport.in;

    location /api/ {
        proxy_pass http://localhost:4000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        rewrite ^/api/(.*)$ /$1 break;
    }

    location / {
        root /var/www/html;
        try_files $uri $uri/ /index.html;
    }
}

5. SAVE: Ctrl+O, Enter, Ctrl+X

6. RELOAD:
sudo nginx -t && sudo systemctl reload nginx

7. TEST:
curl https://mydiamond99clientsupport.in/api/health
